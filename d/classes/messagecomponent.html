    # MessageComponent

    # PROPERTIES


    # .client

    The client who send this button.

    Returns: Client (opens new window)

    # .id

    The custom id of this button (if provided).

    Returns: String (opens new window)

    # .version

    The version of this button.

    Returns: Number (opens new window)

    # .token

    The token of this button interaction.

    Returns: String (opens new window)

    # .discordID

    The id which is provided by Discord for this button.

    Returns: Snowflake (opens new window)

    # .applicationID

    The id of this interaction's application.

    Returns: Snowflake (opens new window)

    # .message

    The message of this button.

    Returns: Message (opens new window)

    # .guild

    The guild of this button.

    Returns: Guild (opens new window)

    # .channel

    The channel of this button.

    Returns: Channel (opens new window)

    # .clicker

    The user who clicked this button.

    Returns: Object<User (opens new window), GuildMember (opens new window), fetch: Promise<boolean>>

    # .webhook

    The webhook of this button interaction.

    Returns: WebhookClient

    # .reply

    Reply for the button

    Returns: InteractionReply
